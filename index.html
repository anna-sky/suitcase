How it works

So, you are impressed with what you see so far and want to know how this actually works? You can watch the video for an overview or you can read on.

Eco-switch is one of the usecases on our technology architecture “The IoT Suitcase”. We have a cloud engine that talks to all devices and manages them. This cloud exposes a real time API, using which a cloud dashboard & a mobile/tablet app interact. All the requests you make from the dashboard or the app are sent to the cloud engine which talks to the device and updates it.
